# SalesSavvy - E-commerce Backend System

## 📌 Description
SalesSavvy is a backend-driven e-commerce application developed using Java and Spring Boot.  
It provides APIs for managing products, users, and orders, enabling core e-commerce functionalities.

---

## 🚀 Tech Stack
- Java
- Spring Boot
- Hibernate (JPA)
- MySQL
- REST APIs
- Postman (for API testing)

---

## ⚙️ Features
- User management (create, update, delete users)
- Product management (add, view, update, delete products)
- Order processing system
- RESTful API design for all operations
- Database integration with MySQL

---

## 🗄️ Database Design
- Designed relational schema for users, products, and orders
- Implemented relationships using Hibernate (JPA)
- Used SQL queries for efficient data handling

---

## 🔗 API Endpoints (Sample)
- `POST /users` → Create user  
- `GET /products` → Get all products  
- `PUT /products/{id}` → Update product  
- `DELETE /orders/{id}` → Delete order  

---

## ▶️ How to Run
1. Clone the repository  
2. Open in IDE (IntelliJ / Eclipse)  
3. Configure MySQL database (update application.properties)  
4. Run Spring Boot application  
5. Use Postman to test APIs  

---

## 📂 Project Structure
- Controller Layer (API handling)
- Service Layer (business logic)
- Repository Layer (database interaction)
- Entity Classes (data models)
